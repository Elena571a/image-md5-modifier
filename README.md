# 图片MD5修改工具

一个用于生成多个不同MD5值的图片版本的工具，适用于需要绕过图片去重检测的场景。

## 功能特点

- ✅ **多种使用方式**：网页版、命令行版、图形界面版
- ✅ 默认生成3个（可自定义数量）不同MD5值的图片版本
- ✅ **组合使用多种方法**：每次随机组合2-4种方法，增强效果
- ✅ 保持图片视觉质量，修改几乎不可见
- ✅ 支持常见图片格式（JPG, PNG, GIF, BMP等）
- ✅ 自动检测并避免MD5重复

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 🌐 网页版（推荐）

1. 双击 `启动网页版.bat` 启动Web服务器
2. 在浏览器中访问 `http://localhost:5001`
3. 上传图片，设置生成数量，点击"开始处理"
4. 下载生成的图片（单个或ZIP打包）

### 💻 命令行版

#### 基本用法

```bash
python image_md5_modifier.py your_image.jpg
```

这会在图片所在目录生成3个不同版本（默认），文件名格式为：`原文件名_v01.jpg`, `原文件名_v02.jpg` 等。

### 指定输出目录

```bash
python image_md5_modifier.py your_image.jpg -o output_folder
```

### 生成指定数量的版本

```bash
python image_md5_modifier.py your_image.jpg -n 5   # 生成5张
python image_md5_modifier.py your_image.jpg -n 10  # 生成10张
```

你可以通过 `-n` 参数指定任意数量的图片，工具会确保每张图片的MD5都不同。

### 完整参数

```bash
python image_md5_modifier.py <输入图片> [-o 输出目录] [-n 版本数量]
```

### 🖥️ 图形界面版

双击 `启动工具.bat` 启动GUI程序，使用图形界面操作。

### 📦 批处理文件

将图片拖到 `process_image.bat` 上即可快速处理。

## 修改方法说明

工具使用8种不同的方法，**每次随机组合2-4种方法**来修改图片：

1. **随机噪声** - 添加微小的随机像素噪声（±3）
2. **亮度调整** - 轻微的亮度变化（±2%）
3. **对比度调整** - 轻微的对比度变化（±2%）
4. **饱和度调整** - 轻微的饱和度变化（±2%）
5. **模糊+锐化** - 极小的模糊然后锐化
6. **锐度调整** - 轻微的锐度变化（±2%）
7. **微旋转** - 小于0.2度的旋转
8. **组合调整** - 组合多种轻微调整

所有修改都经过精心设计，确保：
- MD5值不同
- 视觉质量几乎不受影响
- 文件大小保持合理

## 技术说明

- **网页版**：使用Flask框架，支持拖拽上传、批量下载
- **命令行版**：使用argparse处理参数
- **GUI版**：使用tkinter创建图形界面
- 使用PIL/Pillow进行图片处理
- 使用numpy进行像素级操作
- 使用hashlib计算MD5值
- **组合使用多种方法**：每次随机选择2-4种方法组合，增强MD5差异
- 自动检测并避免MD5重复，最多重试10次确保每张图片MD5都不同
