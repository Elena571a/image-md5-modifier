# 部署到云端服务器 - 让应用24小时可用

## 方案一：使用免费的云平台（推荐，最简单）

### 1. Railway（推荐，免费额度充足）

**步骤：**
1. 访问 https://railway.app
2. 使用GitHub账号登录
3. 点击"New Project" -> "Deploy from GitHub repo"
4. 选择你的代码仓库（需要先上传到GitHub）
5. Railway会自动检测Flask应用并部署

**优点：**
- 免费额度：每月$5，足够使用
- 自动部署，无需配置
- 提供HTTPS域名

### 2. Render（免费，但有限制）

**步骤：**
1. 访问 https://render.com
2. 注册账号
3. 创建新的Web Service
4. 连接GitHub仓库
5. 设置启动命令：`python app.py`
6. 设置端口：5001（或使用环境变量）

**优点：**
- 完全免费
- 自动HTTPS
- 简单易用

**缺点：**
- 免费版15分钟无活动会休眠
- 首次访问需要等待唤醒

### 3. PythonAnywhere（适合Python应用）

**步骤：**
1. 访问 https://www.pythonanywhere.com
2. 注册免费账号
3. 上传代码文件
4. 配置Web应用
5. 设置WSGI文件

**优点：**
- 专为Python设计
- 免费版可用

## 方案二：使用内网穿透（适合不想买服务器的用户）

### 使用 ngrok（最简单）

**步骤：**
1. 访问 https://ngrok.com 注册账号
2. 下载ngrok
3. 获取authtoken
4. 运行：`ngrok http 5001`
5. 会得到一个公网地址，如：`https://xxxx.ngrok.io`

**优点：**
- 完全免费
- 设置简单
- 立即生效

**缺点：**
- 免费版地址每次重启会变化
- 有流量限制

### 使用 frp（自建，更稳定）

需要自己有一台有公网IP的服务器。

## 方案三：购买云服务器（最稳定）

### 推荐平台：
- **阿里云**：https://www.aliyun.com
- **腾讯云**：https://cloud.tencent.com
- **华为云**：https://www.huaweicloud.com

**最低配置：**
- 1核1G内存
- 1M带宽
- 约50-100元/月

**部署步骤：**
1. 购买服务器
2. 安装Python和依赖
3. 上传代码
4. 使用gunicorn运行（生产环境）
5. 配置域名和HTTPS

## 推荐方案

**如果你想要：**
- **最简单**：使用Railway或Render（免费，自动部署）
- **最稳定**：购买云服务器（需要付费，但24小时在线）
- **临时使用**：使用ngrok内网穿透（免费，但地址会变）

## 注意事项

1. **生产环境配置**：部署到云端时，需要修改 `app.py`：
   - 关闭debug模式：`debug=False`
   - 使用生产级WSGI服务器（如gunicorn）
   - 配置环境变量存储敏感信息

2. **文件存储**：云端部署时，上传的文件会存储在服务器上，需要定期清理。

3. **安全性**：建议添加访问限制或认证，避免被滥用。
